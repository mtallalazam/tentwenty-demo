# TenTwenty Demo

This is a demo project. Demo is available at this link <https://spiffy-starburst-f63360.netlify.app/>

## Getting Started

Access the project directory.

Install dependencies.

```bash
npm install
```

Serve with hot reload at <http://localhost:5173>.

```bash
npm run dev
```

### Build

```bash
npm run build
```
